# Calorie Tracker

A weekly calorie tracking PWA built with React + Vite.

## Setup

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

Push to the `main` branch — GitHub Actions will build and deploy automatically.

Your app will be live at: `https://YOUR-USERNAME.github.io/calorie-tracker`

## Notes

- Add icon-192.png and icon-512.png to the `public/` folder for the PWA install icon
- Data is stored in localStorage and resets each week (saved meals persist)
